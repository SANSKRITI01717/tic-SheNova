# Herlytics

A full-stack menstrual health tracking system focused on structured data collection, adaptive user input, and clear insights with practical recommendations.

---

## 1. Overview

Herlytics improves how menstrual health is tracked and understood.

Instead of simple date logging, it:
- Collects structured cycle, symptom, and lifestyle data  
- Adapts questions based on user responses  
- Provides clear insights along with basic recommendations  
- Displays meaningful summaries and trends  
- Generates simple health reports  

The goal is to make tracking more intuitive, relevant, and useful in real-life situations.

## 3. Core Features

### 3.1 Multi-Step Onboarding

A structured step-by-step flow to collect:
- Cycle basics  
- Symptoms  
- Lifestyle factors  
- Past history  
- Doctor accessibility  

### 3.2 Adaptive Input Flow

- Questions dynamically change based on user input  
- Conditional rendering using EJS and JavaScript  
- No fixed/static form structure  

### 3.3 User Dashboard

- Current cycle status  
- Estimated next period  
- Daily logging options  
- Summary and insight cards  

### 3.4 Insights and Recommendations

- Cycle summaries and trends  
- Symptom pattern visibility  
- Basic recommendations based on user inputs  

Focus is on helping users understand their data and take simple actions.

### 3.5 Report Generation

- Cycle overview  
- Symptom history  
- Pattern summaries  

Reports are structured for easy reading and sharing.

### 3.6 Healthcare Support

- Highlights important patterns  
- Suggests when to consider consulting a doctor  
- Allows input for location-based support  

### 3.7 Government Schemes Awareness

- Provides awareness about menstrual health-related government schemes  
- Helps users explore available public health support  

### 3.8 Diet and Exercise Suggestions

- Basic diet recommendations based on cycle phase  
- Simple exercise suggestions aligned with user condition  

### 3.9 FAQ Section

- Answers common menstrual health questions  
- Helps users understand normal vs concerning patterns  
- Designed for quick and accessible information
